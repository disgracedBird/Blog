# Mini Blog

A simple, clean blog platform built with **Django**.

## Features

- User authentication
- Sign up / Login / Logout
- Posts
- Create, edit, delete your own posts
- Rich text support
- Image upload
- Categories
- Create and manage post categories
- Responsive design (Tailwind CSS)
- Clean admin interface
- Comment system under posts

## Currently in progress

- Search & filtering

## Changes

- Fixed the signup form not working
- Moved categories and posts to their own links
- Can now delete and update categories
- Added a side-bar in admin panel to make it easier to navigate
- images are no longer required when making a post/default one is provided

## Tech stack

- Django 5.x
- Python 3.11+
- Tailwind CSS
- SQLite (development)
- django-widget-tweaks (form styling)
